# BabelGames – Biblioteca de Jogos com Avaliações

BabelGames é uma aplicação web que permite aos usuários explorar, registrar e avaliar jogos em uma biblioteca pessoal. O objetivo do projeto é criar uma plataforma simples e intuitiva onde jogadores podem organizar os jogos que conhecem ou jogaram, atribuir notas e escrever reviews.

* Cadastro de jogos (com hx-post)
* Avaliação de jogos com notas
* Remoção ou edição das avaliações (com hx-delete e hx-put)
* Listagem e busca de jogos (com scroll infinito e hx-get)
* Visualização de detalhes e avaliações
